AF Conference management tool

	First of all, you need to install NodeJs to the client environment. For this you can use the following link. Link: https://nodejs.org/en/download/

	Extract the provided Zipped project File.

	Now you need to deploy the backend of the project.

	Step 1 – Open command prompt and navigate to the backend directory. (You can use a preferred              
			 IDE as well like Intellij, WebStorm, VisualStudio Code) Ex: >> cd backend
			 
	Step 2 	-You have to install node modules,
			Using command >> npm install  
         
    Step 3	-You have to run backend project 
			Using command>>npm run start


	After this, the project will be connected to the MongoDB Atlas and Server will be up and running. Then you need to deploy the frontend of the project.

	Step 1 - Open command prompt and navigate to the frontend directory Ex: >> cd {your path}\frontend

	Step 2 – Use these commands >>npm install

	Step 3- You have to run the folder using >>npm start


	After done with the above mentioned steps, the Project frontend will be up and running using specific port numbers.
	Finally you can open the webapplication using a preferred Browser (Ex: Google Chrome)
	
	*In our user guide we have provided the link to the project that we have hosted online
